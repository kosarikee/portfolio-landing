7. ФОРМА ОБРАТНОЙ СВЯЗИ
──────────────────────────

   ✅ Server Action: src/app/actions.ts
      — submitContactForm принимает FormData
      — Валидирует поля (длина, обязательность)
      — Отправляет POST на api.telegram.org/bot{token}/sendMessage
      — Форматирует сообщение в HTML (имя, контакт, задача, дата)
      — Экранирует HTML-спецсимволы (защита от XSS)

   ✅ Клиентская часть (page.tsx → ContactForm)
      — useActionState: idle → загрузка (спиннер) → успех / ошибка
      — При успехе: зелёный блок с сообщением и кнопкой "Отправить ещё"
      — При ошибке: красный блок с конкретным текстом (зависит от кода)
      — Автосброс формы через useEffect после успеха
      — Все поля required + серверная валидация (двойная защита)

   ⚠ Ваш бот должен иметь право писать в чат TELEGRAM_CHAT_ID
      — Если не приходят уведомления: напишите боту /start из целевого чата
